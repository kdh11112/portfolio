parseInt 의 리턴값은 int로 기본자료형(primitve type)

문자열 반환할 경우 기본자료형으로 받고싶을땐 parseInt를 사용 new Integer()로 반환 하고싶을때 valueOf로 사용

int - 기본자료형은 int, float, long, double이 있다.
    -null값으로 초기화 불가능

Integer - Wrapper 클래스
	-null값 처리 가능

Wrapper클래스는 Boxing과 Unboxing이 있다

Primitive Type ->Wrapper class : Boxing
 ex))Integer a = new Integer();

Wrapper class -> Primitve Type : Unboxing
 ex))int b = a.intValue();

돈계산시 float과 double이 아닌 BigDecimal,int long으로 해야한다
9자리가 넘지 않을시 int 18자리가 넘지 않을시 long 18자리 초과시 BigDecimal로 사용 BigDecimal은 정확하나 사용시 불편하고 실행속도가 느리다
BigDecimal을 사용하지 않을시 int나 long 타입은 소숫점을 없앤 형태로 계산해야된다
float형은 7자리 아래 double형은 14자리 아래
